## 📝 Dataset Overview 

* **Data Type:** RGB Images (JPEG/PNG), JSON Annotations (COCO format, OBB, Keypoints), Text files (`pairs.txt`)
* **Size:** 18.91 GB
* **Description:** BECA is a large-scale computer vision dataset designed for dorsal-view beef cattle recognition, comprising two subsets: **BECA-D** for visual diversity (5,661 cattle) and **BECA-L** for long-term tracking (103 cattle over 5 months). It fills the gap in real-world longitudinal cattle re-identification data by providing comprehensive annotations for object detection (Axis-Aligned and Oriented Bounding Boxes), pose estimation (13 keypoints), and individual identity. The dataset serves as a benchmark for precision livestock farming, facilitating research in animal identification, behavior analysis, and welfare monitoring.
